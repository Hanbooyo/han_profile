TRIPLANER 팀프로젝트

  • 팀 프로젝트
  • 프로젝트 수행인원: 총 4인
  • 프로젝트 기간: 2023.05.08 ~ 2023.05.30 (3주)
  • 사용기술: JAVA, SpringBoot, JPA, MySQL, AWS RDS, Google Maps API
  • 데모페이지: TRIPLANER 데모페이지
  • Github Link: https://github.com/Hanbooyo/map-application
  • 프로젝트 설명

    Spring Boot, MySQL, JPA 활용한 여행일정관리 웹사이트입니다.
    Google Maps API를 주로 사용한 지도 + 캘린더 어플리케이션입니다.

    • 구현 기능
    1. 회원가입/탈퇴
      • 회원탈퇴시 비밀번호 확인 후 아이디와 비밀번호가 일치하는 경우 탈퇴진행
    2. 로그인/로그아웃
    3. 캘린더 라이브러리
    4. Google Maps API 마커생성
    5. Google Geocode API 장소정보 호출
    6. 장소정보 DB저장
    7. REST API CRUD (회원가입,캘린더,지도)
  • 해당 프로젝트에서 담당한 부분
    REST API 설계
    로그인 / 로그아웃 (세션)
    JPA와 서비스로직 작성
    데모페이지 호스팅
